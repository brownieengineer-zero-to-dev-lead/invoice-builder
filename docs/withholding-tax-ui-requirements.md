# UI Requirements: ระบบภาษีหัก ณ ที่จ่าย (Withholding Tax System)

---

## 1. Sidebar

เพิ่ม 2 group ใหม่ระหว่าง "เอกสาร" กับ "เทมเพลต"

```
เอกสาร
├── ใบแจ้งหนี้
└── ใบเสนอราคา

พนักงาน              ← group ใหม่
├── ภ.ง.ด.1
└── 50 ทวิ

ผู้รับจ้าง            ← group ใหม่
├── ธุรกรรม
└── รายงานภาษี

เทมเพลต
...

ข้อมูล
├── ลูกค้า
├── พนักงาน          ← เพิ่มใหม่
├── ผู้รับจ้าง        ← เพิ่มใหม่
├── ธุรกิจ
├── ธนาคาร
...
```

## 2. หน้าพนักงาน (Employee List)

### Layout
- เหมือน Client List ที่มีอยู่แล้ว
- ปุ่ม "เพิ่มพนักงาน" มุมขวาบน

### ตาราง

| คอลัมน์ | หมายเหตุ |
|---|---|
| ชื่อ-สกุล | |
| เลขผู้เสียภาษี | |
| เงินเดือนตั้งต้น | |
| action | edit / delete |

### Form (เพิ่ม / แก้ไข)

```
ชื่อ-สกุล *
เลขบัตรประชาชน / เลขผู้เสียภาษี *
ที่อยู่ *
เงินเดือนตั้งต้น
```

## 3. หน้าผู้รับจ้าง (Contractor List)

### Layout
- เหมือน Client List ที่มีอยู่แล้ว
- ปุ่ม "เพิ่มผู้รับจ้าง" มุมขวาบน

### ตาราง

| คอลัมน์ | หมายเหตุ |
|---|---|
| ชื่อ / ชื่อบริษัท | |
| ประเภท | บุคคลธรรมดา / นิติบุคคล |
| เลขผู้เสียภาษี | |
| action | edit / delete |

### Form (เพิ่ม / แก้ไข)

```
ประเภท *            [บุคคลธรรมดา / นิติบุคคล]
ชื่อ / ชื่อบริษัท *
เลขบัตรประชาชน / เลขทะเบียนนิติบุคคล *
ที่อยู่ *
```

## 4. หน้า ภ.ง.ด.1

### List Page

**Filter bar**
```
พนักงาน (dropdown)  |  เดือน (dropdown)  |  ปี (dropdown)
```

**ตาราง**

| คอลัมน์ | หมายเหตุ |
|---|---|
| พนักงาน | |
| เดือน / ปี | |
| เงินได้ | |
| ภาษีที่หัก | |
| action | edit / delete / export PDF |

### Form Page

```
┌─────────────────────────────────────┐
│  เดือน / ปี *          [MM / YYYY]  │
│                                     │
│  พนักงาน *             [dropdown]   │
│                                     │
│  เงินได้ *             [input]      │
│  (default = เงินเดือนตั้งต้น)       │
│                                     │
│  ภาษีที่หัก *          [input]      │
│  (คำนวณอัตโนมัติ แก้ได้)           │
│                                     │
│         [บันทึก]  [Preview]         │
└─────────────────────────────────────┘
```

**หมายเหตุ:** ข้อมูล autofill ที่แก้ไม่ได้ (ชื่อ / เลขภาษี / ที่อยู่ของพนักงาน) จะแสดงใน Preview เท่านั้น

**Validation**
- ถ้ามี record ของพนักงานคนนี้ในเดือนนี้แล้ว → warning "มีข้อมูลเดือนนี้แล้ว ต้องการแก้ไขหรือไม่?"

## 6. หน้าธุรกรรม (ผู้รับจ้าง)

### List Page

**Filter bar**
```
ผู้รับจ้าง (dropdown)  |  ประเภท (ภ.ง.ด.3 / 53)  |  เดือน  |  ปี
```

**ตาราง**

| คอลัมน์ | หมายเหตุ |
|---|---|
| ผู้รับจ้าง | |
| ประเภท | ภ.ง.ด.3 / ภ.ง.ด.53 (autofill จากประเภทผู้รับจ้าง) |
| วันที่จ่าย | |
| จำนวนเงินก่อนหัก | |
| ภาษีที่หัก | |
| action | edit / delete / export 50 ทวิ |

### Form Page

```
┌─────────────────────────────────────┐
│  ธุรกิจ *              [dropdown]   │
│                                     │
│  ผู้รับจ้าง *          [dropdown]   │
│                                     │
│  วันที่จ่าย *          [datepicker] │
│                                     │
│  ประเภทเงินได้ *       [dropdown]   │
│  ค่าบริการ / ค่าเช่า / ค่าสิทธิ์   │
│                                     │
│  อัตรา WHT *           [dropdown]   │
│  1% / 3% / 5% / 15%                │
│  (autofill จากประเภทเงินได้ แก้ได้) │
│                                     │
│  อ้างอิง Invoice       [dropdown]   │
│  (optional — autofill ยอดเงิน)      │
│                                     │
│  จำนวนเงินก่อนหัก *   [input]      │
│  ภาษีที่หัก            (คำนวณอัตโนมัติ, แก้ได้) │
│                                     │
│  วิธีนำส่ง *           [dropdown]   │
│                                     │
│         [บันทึก]  [Preview]         │
└─────────────────────────────────────┘
```

**หมายเหตุ:** ข้อมูล autofill ที่แก้ไม่ได้ (ชื่อ / เลขภาษี / ที่อยู่ ของธุรกิจและผู้รับจ้าง) จะแสดงใน Preview เท่านั้น

### Export 50 ทวิ

กด export จากรายการในหน้า List หรือหลัง save

```
[export ฉบับที่ 1-2 สำหรับผู้รับจ้าง]
[export ฉบับที่ 3-4 สำหรับภายใน]
```

## 7. หน้ารายงานภาษี (ผู้รับจ้าง)

### List Page

**Filter bar**
```
เดือน (dropdown)  |  ปี (dropdown)  |  ประเภท (ภ.ง.ด.3 / 53)
```

**ตาราง** (แสดงสรุปรายเดือน)

| คอลัมน์ | หมายเหตุ |
|---|---|
| เดือน / ปี | |
| ประเภท | ภ.ง.ด.3 / ภ.ง.ด.53 |
| จำนวนรายการ | นับจากธุรกรรม |
| ยอดรวมเงินได้ | |
| ยอดรวมภาษีที่หัก | |
| action | ดูรายละเอียด / export PDF |

### Detail Page

```
┌─────────────────────────────────────┐
│  ธุรกิจ *              [dropdown]   │
│  เดือน / ปี *          [MM / YYYY]  │
│  ประเภท                ภ.ง.ด.3      │
│                                     │
│  ยอดรวมเงินได้         (คำนวณจากธุรกรรม) │
│  ยอดรวมภาษีที่หัก      (คำนวณจากธุรกรรม) │
│                                     │
│         [export ภ.ง.ด. PDF]         │
└─────────────────────────────────────┘
```

**หมายเหตุ:** PDF ที่ export เป็นเฉพาะหน้า Summary — รายละเอียด transaction แต่ละรายการดูได้จากระบบกรมสรรพากรโดยตรง

---

## 8. PDF Output

| Feature | รูปแบบ PDF |
|---|---|
| ภ.ง.ด.1 | ตารางสรุปรายเดือน ทุกพนักงาน |
| 50 ทวิ พนักงาน | ใบรับรองต่อคน ตามรูปแบบกรมสรรพากร |
| ธุรกรรม → 50 ทวิ ฉบับ 1-2 | ใบรับรองสำหรับผู้รับจ้าง |
| ธุรกรรม → 50 ทวิ ฉบับ 3-4 | ใบรับรองสำหรับภายใน |
| รายงานภาษี | ภ.ง.ด.3 / 53 สรุปรายเดือน |
