## Master Data ที่ต้องเพิ่ม

### พนักงาน (entity ใหม่)

|---|---|
| ชื่อ-สกุล | required |
| เลขบัตรประชาชน / เลขผู้เสียภาษี | required |

| ฟิลด์ | รายละเอียด |

| เลขบัตรประชาชน / เลขทะเบียนนิติบุคคล | required, ขึ้นกับประเภท |
| ที่อยู่ | required |

## Feature 1: ภ.ง.ด.1

**วัตถุประสงค์**
บันทึกการจ่ายเงินเดือนและภาษีที่หักต่อพนักงานรายเดือน เพื่อใช้เป็นข้อมูลยื่นกรมสรรพากรและออก 50 ทวิ

### ข้อมูลต่อ Record

| ฟิลด์ | รายละเอียด |
|---|---|
| เดือน / ปี | required |
| พนักงาน | autofill ชื่อ / เลขภาษี / ที่อยู่ |
| เงินได้ | default = เงินเดือนตั้งต้น, แก้ได้ |
| ภาษีที่หัก | คำนวณให้อัตโนมัติ, แก้ได้ |

### หน้า List

- Filter: พนักงาน / เดือน / ปี
- แสดง: พนักงาน / เดือน / เงินได้ / ภาษีที่หัก
- Export PDF สรุปรายเดือน

### Business Rules

- 1 พนักงาน ต่อ 1 เดือน มีได้ 1 record เท่านั้น
- ถ้าเดือนนั้นมี record อยู่แล้ว → แจ้งเตือนก่อน edit
- เงินได้รวมทุกประเภท (เงินเดือน / โบนัส / OT) เป็นตัวเลขเดียว

---

## Feature 2: 50 ทวิ (พนักงาน)

**วัตถุประสงค์**
ออกหนังสือรับรองการหักภาษี ณ ที่จ่ายให้พนักงานรายปี โดยดึงข้อมูลจาก ภ.ง.ด.1

### ข้อมูลต่อ Record

| ฟิลด์ | รายละเอียด |
|---|---|
| ปีภาษี | required |
| พนักงาน | autofill ชื่อ / เลขภาษี / ที่อยู่ |
| รายการเงินได้ | ดึงจาก ภ.ง.ด.1 ทั้งปี, แก้ / เพิ่ม manual ได้ |
| เงินได้รวม | คำนวณอัตโนมัติ |
| ภาษีที่หักรวม | คำนวณอัตโนมัติ |
| วิธีนำส่ง | หัก ณ ที่จ่าย / ออกให้ตลอดไป / ออกให้ครั้งเดียว |

### หน้า List

- Filter: พนักงาน / ปีภาษี
- แสดง: พนักงาน / ปี / เงินได้รวม / ภาษีรวม
- Export PDF ต่อใบ

### Business Rules

- 1 พนักงาน ต่อ 1 ปีภาษี มีได้ 1 record
- ถ้า ภ.ง.ด.1 ไม่ครบ 12 เดือน → แจ้งเตือน แต่ยังสร้างได้
- เพิ่มรายการ manual ได้กรณีมีเงินได้พิเศษนอกระบบ

---

## Feature 3: ธุรกรรม (ผู้รับจ้าง)

**วัตถุประสงค์**
บันทึกรายการจ่ายเงินให้ผู้รับจ้างรายครั้ง เพื่อออก 50 ทวิ และรวมเป็นรายงานภาษีรายเดือน

### ข้อมูลต่อ Record

| ฟิลด์ | รายละเอียด |
|---|---|
| ธุรกิจ | required, autofill ชื่อ / เลขภาษี / ที่อยู่ |
| วันที่จ่าย | required |
| ผู้รับจ้าง | autofill ชื่อ / ประเภท / เลขภาษี / ที่อยู่ |
| ประเภท ภ.ง.ด. | auto จากประเภทผู้รับจ้าง — บุคคลธรรมดา → ภ.ง.ด.3, นิติบุคคล → ภ.ง.ด.53 |
| ประเภทเงินได้ | ค่าบริการ / ค่าเช่า / ค่าสิทธิ์ / อื่นๆ |
| อัตรา WHT | 1% / 3% / 5% / 15% |
| จำนวนเงินก่อนหัก | กรอกเอง หรือดึงจาก Invoice (optional) |
| ภาษีที่หัก | คำนวณอัตโนมัติ |
| วิธีนำส่ง | หัก ณ ที่จ่าย / ออกให้ตลอดไป / ออกให้ครั้งเดียว |

### Export 50 ทวิ (4 ฉบับ)

| ฉบับ | สำหรับ |
|---|---|
| ฉบับที่ 1-2 | ผู้รับจ้าง |
| ฉบับที่ 3-4 | ภายใน |

### หน้า List

- Filter: ผู้รับจ้าง / ประเภท ภ.ง.ด. / เดือน / ปี
- แสดง: ผู้รับจ้าง / วันที่ / ประเภท / ยอด / ภาษีที่หัก
- Export 50 ทวิ ต่อรายการ

---

## Feature 4: รายงานภาษี (ผู้รับจ้าง)

**วัตถุประสงค์**
รวมยอดธุรกรรมทั้งเดือน เพื่อ export เป็นเอกสาร ภ.ง.ด.3 / ภ.ง.ด.53 (เฉพาะหน้า Summary) สำหรับยื่นกรมสรรพากร — รายละเอียด transaction แต่ละรายการบัญชีดูได้จากระบบของกรมสรรพากรโดยตรง

### ข้อมูล (ดึงจากธุรกรรม ไม่มี record แยก)

| ฟิลด์ | รายละเอียด |
|---|---|
| ธุรกิจ | required |
| เดือน / ปี | required |
| ประเภท ภ.ง.ด. | ภ.ง.ด.3 / ภ.ง.ด.53 |
| ยอดรวมเงินได้ | คำนวณจากธุรกรรมในเดือนนั้นอัตโนมัติ |
| ยอดรวมภาษีที่หัก | คำนวณจากธุรกรรมในเดือนนั้นอัตโนมัติ |

### Business Rules

- รวมเฉพาะธุรกรรมในเดือน / ปี ที่เลือก
- แยก export ภ.ง.ด.3 และ ภ.ง.ด.53 ออกจากกัน
- Export เฉพาะหน้า Summary — ไม่มีใบแนบรายละเอียด

## ความสัมพันธ์ทั้งหมด

```
พนักงาน (master)   → ภ.ง.ด.1 (รายเดือน)      → 50 ทวิ พนักงาน (รายปี)

ผู้รับจ้าง (master) → ธุรกรรม (รายครั้ง)      → export 50 ทวิ 4 ฉบับ
                            ↓
                     รายงานภาษี (รายเดือน)     → export ภ.ง.ด.3 / 53

Invoice (optional)  → ธุรกรรม
```
